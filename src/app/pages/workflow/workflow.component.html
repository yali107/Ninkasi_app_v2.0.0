Workflow

The entire project last for 3 weeks.

  • I. Web scraping

    280,000 beer reviews, info and ratings are scraped from ratebeer.com using Python Scrapy package

  • II. Data Cleaning

    Beer info and rating data are cleaned. Text mining, tokenization, spell check and lemmatization are performed on beer reviews

  • III. Content-based

    Term Frequencyâ€“Inverse Document Frequency (TF-IDF) are calculated on document-term matrix, Latent Semantic Indexing (LSI) is applied, and cosine similarity matrix is generated.

  • IV. Collaborative Filtering

    Two algorithms are applied. A Matrix factorization method-SVD++ and an unsupervised deep learning technique-Restricted Boltzmann Machine (RBM). The results from two methods are combined to reconstruct user-item matrix.
